# RunAnywhere Agent

AI coding assistant specifically designed for building apps with RunAnywhere SDKs.

## Supported SDKs

- **Swift SDK** (iOS/macOS) - `runanywhere-swift`
- **Kotlin SDK** (Android) - `runanywhere-kotlin`
- **React Native SDK** - `@runanywhere/core`, `@runanywhere/llamacpp`
- **Flutter SDK** - `runanywhere`, `runanywhere_llamacpp`

## Installation

```bash
pip install runanywhere-agent
```

## Usage

```bash
runanywhere-agent
```

On first run, you'll be prompted to enter your authorized email address.

## Features

- Pre-configured with RunAnywhere SDK documentation
- Optimized prompts for mobile/cross-platform development
- Support for Claude and GPT models
- Built on top of [aider](https://aider.chat)

## Requirements

- Python 3.10-3.12
- Authorized email (contact founders@runanywhere.ai for access)
